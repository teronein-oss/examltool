## ★ 출력 절대 규칙
1. 아래 출력 형식 이외의 내용은 절대 출력하지 말 것
2. PASSAGE에는 영어 지문만 출력. 한국어 번역·해석·직독직해 절대 금지
3. 머리말·꼬리말·주석·추가 설명 일체 금지
4. EXPLANATION은 한국어로 작성
② 필수 출력 섹션 (유형에 맞게 사용)
PASSAGE:      ← 반드시 포함
DIRECTION:    ← 반드시 포함
CHOICES:      ← 반드시 포함
ANSWER:       ← 반드시 포함
EXPLANATION:  ← 반드시 포함
SUMMARY:      ← 요약문 유형만
GIVEN_SENTENCE: ← 문장삽입만
INTRO/BLOCK_A/B/C: ← 순서 유형만
💡 정답 번호 분산 팁: 프롬프트에 "ANSWER: [①~⑤ 중 하나 — 매 문항마다 다른 번호 선택]" 을 추가하면 정답이 ①에 몰리는 현상을 방지할 수 있습니다.

📖 Seum Teachers Lab 사용 가이드
💡 처음 사용하신다면 아래 순서대로 따라하세요!
1
API 키 발급 필수
상단 드롭다운에서 사용할 모델을 먼저 선택하세요.
Gemini 사용 시 → aistudio.google.com 에서 무료 발급
Claude 사용 시 → platform.claude.com 에서 발급 (유료)
발급한 키를 상단 입력창에 붙여넣으면 ✓ 표시가 나타납니다.
2
지문 입력 필수
📝 지문 입력 탭으로 이동 → + 지문 추가 버튼 클릭
영어 지문을 붙여넣고 저장하세요. 여러 지문을 한번에 추가할 수 있습니다.
자주 쓰는 지문 묶음은 💾 세트 저장으로 이름을 붙여 저장해두면 다음에 바로 불러올 수 있습니다.
3
출제 방식 선택
랜덤 배정 ON: 유형별 몇 문항을 만들지 숫자로 설정하면 지문이 자동 배분됩니다.
랜덤 배정 OFF: 각 지문 카드에서 원하는 유형을 직접 선택할 수 있습니다.
서술형을 포함하려면 하단 서술형 포함 토글을 켜세요.
4
시험지 생성
🚀 시험지 1세트 생성 버튼을 누르면 자동으로 문항이 생성됩니다.
생성 후 문제만 복사 / 해설만 복사 버튼으로 한글·워드에 바로 붙여넣기 하세요.
5
오류 발생 시 해결 방법
❌ Failed to fetch로컬 파일(file://)에서는 작동 안 됩니다. 반드시 https:// 주소로 접속하세요.
❌ Quota exceeded / 한도 초과오늘 API 사용 한도가 소진됐습니다. 내일 다시 시도하거나 다른 모델로 변경하세요.
❌ Failed to fetch (네트워크)인터넷 연결을 확인하세요.
❌ 문제에 해설이 같이 나옴AI가 출력 형식을 지키지 않은 경우입니다. 다시 생성해보세요.
